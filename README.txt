---------------------Installation----------------------
(make sure you are a root user)
cd
git clone https://github.com/arismelachroinos/lscript.git
cd lscript
chmod +x install.sh
./install.sh
-----------------How to run the script-----------------
(make sure you are a root user)
open terminal
type  "l"
press enter
(Not even "lazy"!! Just "l"! The less you type , the better!)
--------------------How to uninstall-------------------
cd /root/lscript
./uninstall.sh
rmdir -r /root/lscript
